## QR Code Generator

sebuah program dengan nama qr code generator yang dibuat dengan bahasa pemrograman javascript. program ini akan mengenerate sebuah QR Code sesuai isi dari value sebuah input.

### dibuat dengan berikut

* HTML
* CSS
  * [Bootstrap](https://getbootstrap.com)
* Javascript

### lainnya

* [API](https://goqr.me/api/)
* handphone
* code editor (Acode)
* edit gambar (Pixellab)
